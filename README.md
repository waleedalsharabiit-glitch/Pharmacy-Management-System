# 🏥 Pharmacy Management System (نظام إدارة الصيدلية الذكي)

نظام ويب متطور لإدارة الصيدليات ومستودعات الأدوية مصمم خصيصاً لتسهيل عملية صرف وتوفير الأدوية للمستفيدين، مبرمج باستخدام **PHP (PDO)** و **MySQL** مع واجهة مستخدم إبداعية وعصرية تعتمد على نمط التصميم الزجاجي (Glassmorphism).

---

## ✨ المميزات الرئيسية للمشروع (Features)

* **لوحة تحكم ذكية (Premium Dashboard):** ترحيبية وتفاعلية للمستفيدين تعرض الإحصائيات الفورية لطلبات الصرف، طلبات التوفير، وآخر الحركات النشطة.
* **البحث واستعراض الأدوية:** إمكانية البحث المتقدم الفوري في مخزن الأدوية لمعرفة الكميات المتاحة والأسعار.
* **طلب صرف الأدوية:** نموذج لطلب صرف الدواء وتحديد الكمية مع إمكانية إرفاق صورة للوصفة الطبية المعتمدة ومراجعتها برمجياً.
* **طلب توفير الأدوية الناقصة:** قسم مخصص للمستفيدين لطلب توفير الأدوية غير المتوفرة بالمخزن، ومتابعة حالة توفيرها خطوة بخطوة عبر مسار بصري تفاعلي (Progress Line).
* **تصميم عصري (Glassmorphism UI):** واجهات مستجيبة (Responsive) متطابقة مع كافة الشاشات مجهزة بتأثيرات بصرية مميزة بدون استخدام أي إطارات خارجية مثل Tailwind لضمان سرعة التحميل القصوى للواجهة.

---

## 🛠️ التقنيات المستخدمة (Tech Stack)

* **Backend:** PHP 8.x (Using PDO for database security and Prepared Statements)
* **Database:** MySQL
* **Frontend:** Vanilla HTML5, Custom CSS3 (Neumorphism / Glassmorphism), Vanilla JavaScript
* **Server Environment:** XAMPP / WampServer (Apache)
